# 🍃 Esan Foods – Healthy Traditional Food Delivery

A modern **food ordering web application** built using **Next.js** that allows users to browse, search, and order healthy homemade foods.

---

## 🚀 Live Demo

👉 https://esan-foods.vercel.app

---

## 📌 Features

* 🛒 Add to Cart with quantity management
* 🔍 Search products instantly
* 📱 Mobile responsive design
* 🍛 Beautiful Hero section UI
* 📦 Dynamic product listing from API
* 📍 Location & Contact pages
* 🍔 Hamburger menu (mobile navigation)
* ⚡ Fast performance with Next.js

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React
* **Styling:** CSS / Inline Styles
* **Backend API:** Node.js (Hosted on Render)
* **Deployment:** Vercel
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/esan-foods.git
cd esan-foods
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
